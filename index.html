Cardboard

Cardboard =
Fabric + Paper

Cardboard is a Fabric mod that adds support for the popular Bukkit/Spigot/Paper plugin API. 
This mod lets you use plugins that are made for Bukkit and it's derivatives (Spigot & Paper) with Fabric.

Cardboard is still in development and will not support every plugin.
Test Server kindly provided by Apex Hosting: cardboardmod.apexmc.co
